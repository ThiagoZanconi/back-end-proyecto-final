# Consideraciones:
Si se cambia la carpeta de instalacion de modelos de Ollama, se tendra que cambiar el valor en el registro para que el programa la encuentre.
Ej. comando windows:
setx OLLAMA_MODELS "D:\.Ollama\models" /M
Ej. comando linux:
export OLLAMA_MODELS="D:\.Ollama\models"
